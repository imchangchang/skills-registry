# AI API 集成 - 更新历史

## 1.2.0 (2026-02-12)

### 新增
- **[强制] Token 统计与成本监控原则**
  - 所有 AI API 调用必须记录 Token 使用量
  - 日志中必须明确打印输入/输出 Token 数量
  - Token 效率作为测试指标之一
  - 提供成本估算示例代码

### 文档更新
- SKILL.md 新增 "Token 统计与成本监控" 章节
- AI 助手约定中新增 Token 统计强制要求

## 1.1.0 (2026-02-11)

### 重大变更
- **重构 Prompt 管理为方法论**：将具体 Prompt 内容从 Skill 中移除
  - Prompt 属于项目业务逻辑，应在项目中维护
  - Skill 只提供管理方法论和工具参考实现

### 新增
- 创建 `templates/ai-project/` 项目模板
  - 包含 Prompt 文件组织结构示例
  - 包含加载器工具类
  - 包含使用示例

### 文档更新
- SKILL.md 重写，强调工具和方法论而非具体 Prompt
- 添加项目模板使用说明
